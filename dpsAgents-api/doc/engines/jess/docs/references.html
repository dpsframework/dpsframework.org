20. For More Information...

20.1. ... about Jess
  • Friedman-Hill, Ernest, "Jess in Action: Rule-based Systems in Java", ISBN 1930110898.
  • The Jess FAQ at http://www.jessrules.com/FAQ.shtml includes the solutions to some common problems.
  • You can post Jess or Java programming questions to StackOverflow. We monitor the jess tag on StackOverflow (so be sure to tag your question appropriately!) and we can answer your questions quickly there.

  • Historically, the jess-users mailing list, was the primary means of asking questions and getting help with Jess. Patches and announcements are still posted to this list, but we no longer encourage people to use this list to ask questions.

    To subscribe, send mail to majordomo@sandia.gov containing the text

        subscribe jess-users
    end

    as the body of the message. Our list is moderated, so it's spam and virus-free.

  • Bug reports.

    No software is ever perfect. Despite our best efforts and extensive testing, there may still be bugs in Jess. Please read the release notes for specific information. Comments and bug reports are welcome. Contact me at ejfried@sandia.gov so I can fix them for a later release.
